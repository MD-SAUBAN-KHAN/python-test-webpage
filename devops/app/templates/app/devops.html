DevOps and Related Concepts

DevOps

DevOps is a set of practices that automates the processes between software development and IT teams.

DevOps Tools

  • Ansible
  • Terraform
  • Docker
  • Kubernetes
  • Observability
  • Shell Scripts
→

Ansible

Ansible is an open-source automation tool used for configuration management, application deployment, and orchestration.

→

Terraform

Terraform is an open-source tool for building, changing, and versioning infrastructure safely and efficiently.

→

Docker

Docker is a platform for building, running, and shipping applications in lightweight containers.

→

Kubernetes

Kubernetes (K8s) automates the deployment, scaling, and management of containerized applications.

→

Observability

Observability refers to the ability to measure a system's internal states through logs, metrics, and traces.

→

Shell Scripts

Shell scripting is a powerful way to automate repetitive tasks in Unix/Linux environments.
